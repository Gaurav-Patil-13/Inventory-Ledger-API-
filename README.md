# Inventory Ledger API

# Install Dependencies

Run this command in the project root directory:

```bash
npm install
```
# Run Application

Development Mode:

```bash
npm run dev
```


# Server URL

Application runs on:

```text
http://localhost:3000
```

Frontend UI:

```text
http://localhost:3000
```

Entries API:

```text
http://localhost:3000/entries
```

Summary API:

```text
http://localhost:3000/summary
```
